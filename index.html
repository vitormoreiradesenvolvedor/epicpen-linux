Funcionalidades

Tudo o que precisa, no Linux

As mesmas ferramentas que tornaram o EpicPen popular no Windows, agora com suporte nativo a Wayland e integração total com ambientes Linux modernos.

✏️

Caneta e Marcador

Desenhe sobre o ecrã com traço suave. Múltiplas cores e espessuras ajustáveis em tempo real.

🔷

Formas Geométricas

Retângulos, elipses e setas para estruturar anotações de forma rápida durante apresentações.

↩️

Desfazer / Refazer

Histórico completo de ações com Ctrl+Z e Ctrl+Y. Sem medo de erros durante a sessão.

🔴

Ponteiro Laser

Realce visualmente qualquer área do ecrã com um ponto vermelho luminoso para audiências.

🔍

Lupa / Zoom

Lupa circular com ampliação configurável. 15 fps em Wayland, 60 fps em X11.

⬜

Quadro Branco

Sobreponha um fundo branco para usar como quadro virtual sem sair do ambiente gráfico.

🖥️

Múltiplos Monitores

Suporte nativo a configurações multi-monitor com overlay independente por ecrã.

⌨️

Atalhos Globais

Ative e desative ferramentas com atalhos de teclado globais sem interromper o fluxo.

📌

Toolbar Flutuante

Toolbar colapsável e sempre visível no topo, que não interfere com outras janelas.

Por que EpicPen Linux?

O EpicPen original é Windows only

  • Nativo em Wayland — usa wlr-layer-shell para overlay sem gambiarras
  • Sem instalação — AppImage único, executa em qualquer distro Linux x86_64
  • Python bundlado — Python 3.12 + PyQt6 incluídos, sem conflito com o sistema
  • Open-source MIT — código aberto, auditável, melhorável pela comunidade
  • Compatível com distros antigas — funciona a partir de Ubuntu 22.04 / glibc 2.34
  • Tray icon — integra com a system tray para acesso rápido sem janela permanente

Instalação

Pronto em 30 segundos

Não é necessário instalar nada. Descarregue, dê permissão de execução e corra.

🐧

EpicPen Linux AppImage

Todas as dependências bundladas. Funciona em Ubuntu, Fedora, Arch, Debian, Manjaro e mais.

# Descarregar o AppImage mais recente
wget https://github.com/vitormoreiradesenvolvedor/epicpen-linux/releases/latest/download/EpicPen-latest-x86_64.AppImage

# Dar permissão de execução
chmod +x EpicPen-*-x86_64.AppImage

# Executar
./EpicPen-*-x86_64.AppImage
